## Frequently Asked Questions

**How do I set a custom alarm message?**
Open the app, create a new alarm, and enter your custom message before saving.

**My alarm didn’t go off — what should I do?**
Make sure notifications are enabled and your device is not on silent or Focus mode.

**Can I change my message later?**
Yes, you can edit any alarm at any time.
